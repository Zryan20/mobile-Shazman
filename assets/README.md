## ğŸ“ Asset Guidelines

### Image Guidelines
- **Format:** PNG for icons/UI, JPEG for photos
- **Transparency:** Use PNG with alpha channel when needed
- **Compression:** Optimize for mobile (use tools like TinyPNG)
- **Naming:** Use lowercase with underscores (e.g., `lesson_icon_01.png`)
- **Size:** Keep under 500KB per image

### Audio Guidelines
- **Clarity:** Ensure clear pronunciation for learning
- **Consistency:** Maintain consistent volume levels
- **Format:** MP3 for compatibility, AAC for quality
- **Compression:** Balance quality and file size
- **Silence:** Trim leading/trailing silence

### Cultural Considerations
- All visual content should be culturally appropriate for Kurdish learners
- Avoid culturally sensitive imagery
- Use inclusive, diverse character representations
- Respect religious and cultural sensitivities
